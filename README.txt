=====================================================
WIZ PUNCH
=====================================================

WIP

Wolfenstein 3D inspired Dungeon Crawler
FPS. Built with Python (PyGame).

Project Based On: https://youtu.be/ECqUrT7IdqQ


----------------------------------------------------

Setup:
	1. Install Python & git clone this repo 
	2. Setup virtual environment (venv) with
		python3 -m venv <virtual env name>
	3. Activate the venv and pip install pygame
	4. Run the game with python3 main.py
